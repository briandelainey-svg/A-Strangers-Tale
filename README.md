# A-Strangers-Tale
A TABG(Text Based Adventure Game) I'm using for my Comp-Sci class.

External files hold a set of data that won’t be deleted when ending a program. A file like this can be used for save data for games, like an inventory and names, to store highscores from previous players, or to simply store user settings. Personally I used it for saving the players name and inventory, as well as story events.
